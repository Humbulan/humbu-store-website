What is Coding?

Coding (or programming) is the process of creating instructions for computers to follow. Think of it like writing a recipe, but instead of making cookies, you're making apps, websites, games, and more!

The building blocks of all programs

1. Variables - Storing Information

Variables are like labeled boxes where you store information. You can put different types of data in them and use them later!

üêç Python Examples
# Storing different types of information
name = "Alex" # Text (string)
age = 16 # Number (integer)
height = 5.8 # Decimal (float)
is_student = True # True/False (boolean)

2. Functions - Reusable Code Blocks

Functions are like mini-programs that do specific tasks. You can use them over and over again!

üêç Function Example

    return f"Hello, {name}! Welcome to coding!"

# Using the function

print(message) # Output: Hello, Sarah! Welcome to coding!

3. Loops - Repeating Actions

Loops let you repeat code multiple times without writing it over and over. Perfect for repetitive tasks!

üîÑ Loop Examples
# Count from 1 to 5
for i in range(1, 6):
    print(f"Count: {i}")

# Repeat while condition is true
countdown = 3
while countdown > 0:
    print(countdown)
    countdown -= 1

Project-Based Learning

The best way to learn programming is by building real projects! Here are beginner-friendly projects that will teach you essential skills:

üéØ Project 1: Number Guessing Game

What you'll learn: Variables, loops, conditionals, user input

Description: Create a game where the computer picks a random number and the player tries to guess it!

üêç Python Version
import random

secret_number = random.randint(1, 100)
guesses = 0

while True:
    guess = int(input("Guess a number (1-100): "))
    guesses += 1

    if guess == secret_number:
        print(f"Congratulations! You got it in {guesses} tries!")
        break
    elif guess < secret_number:
        print("Too low! Try higher.")
    else:
        print("Too high! Try lower.")

    for i, task in enumerate(todo_list, 1):
